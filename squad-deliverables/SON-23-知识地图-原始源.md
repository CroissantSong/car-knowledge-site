# 汽车技术通识 · 六层知识地图

> 面向非汽车专业应届生，从「一窍不通」到「能听懂行业对话」的内容骨架。
> 每层包含：关键知识点（5–10个）、前置依赖、学习目标。

---

## 总览：六层依赖关系图

```
第1层 整车认知 ◄── 零基础入口
   │
   ▼
第2层 机械基础 ◄── 第1层
   │
   ▼
第3层 传统系统 ◄── 第1层 + 第2层
   │
   ├──────────────────────────────┐
   ▼                              ▼
第4层 新能源 ◄── 第2层 + 第3层   第5层 智能汽车 ◄── 第2层 + 第3层 + 第4层部分
   │                              │
   └──────────┬───────────────────┘
              ▼
        第6层 车企工作语境 ◄── 全部前5层（了解全貌后再讲工作流）
```

**学习路径建议：**
- 零基础 → 第1层 → 第2层 → 第3层 → 第4层 / 第5层可并行 → 第6层收尾
- 有工科背景（学过力学/电工）→ 可从第2层直接切入，补第1层术语即可
- 第4层与第5层存在交叉依赖（三电知识影响对智能汽车电气架构的理解），建议先完成第4层再进第5层

## 第1层：整车认知

> **定位**：建立对"汽车"这个对象的整体认知，理解车辆分类、基本参数和常用术语。
> **预计学时**：4–6 小时

### 关键知识点

| # | 知识点 | 简介 |
|---|--------|------|
| 1 | **汽车定义与分类** | 乘用车/商用车，轿车/SUV/MPV/皮卡，A/B/C/D级车划分标准（轴距+排量），中国乘用车分类标准（GB/T 3730.1） |
| 2 | **整车基本结构** | 车身（承载式/非承载式）、底盘、动力总成、电气设备 — 四大组成部分及各自职责 |
| 3 | **车辆坐标系与基本尺寸** | 轴距、轮距、长宽高、接近角/离去角、最小离地间隙 — 以「人的视角」理解各参数含义 |
| 4 | **VIN码与车辆身份证** | 车辆识别代号（VIN）的结构：WMI（世界制造商识别码）+ VDS（车辆说明）+ VIS（车辆指示）三段 |
| 5 | **驱动形式** | FWD（前驱）、RWD（后驱）、AWD/4WD（四驱）— 各形式的优缺点与常见车型 |
| 6 | **车身类型与平台** | 三厢/两厢/掀背/旅行/轿跑/敞篷；平台化概念入门（MQB/TNGA/CMA 等） |
| 7 | **汽车发展简史** | 1886年奔驰一号 → 福特流水线 → 丰田精益生产 → 电动化/智能化浪潮（中国新能源崛起时间线） |
| 8 | **常用术语扫盲** | 马力/扭矩、排量、燃油标号、整备质量/满载质量、工信部油耗/NEDC/WLTC/CLTC 工况 |

### 前置依赖

- 无硬性前置。本层是零基础入口。

### 学习目标

学完本层后，学员能够：
1. 在停车场/展厅中准确说出 5 种以上常见车型的分类名称
2. 读懂一张汽车参数表（尺寸、动力、驱动形式），并解释每个参数的实际意义
3. 看到一辆车的 VIN 码，指出其产地国和制造商
4. 用 3 分钟向朋友讲清楚「汽车的基本构造」

## 第2层：机械基础

> **定位**：为理解汽车各子系统（发动机、底盘等）提供必要的力学、材料和热力学基础知识。
> **预计学时**：8–10 小时（工科背景可压缩至 4 小时）

### 关键知识点

| # | 知识点 | 简介 |
|---|--------|------|
| 1 | **力与运动** | 牛顿三定律在汽车中的体现：加速（F=ma）、制动（惯性力）、转向（向心力） |
| 2 | **摩擦与润滑** | 摩擦类型（滑动/滚动）、摩擦系数、润滑原理（油膜形成），在轮胎抓地/发动机润滑中的应用 |
| 3 | **材料力学入门** | 应力-应变、屈服强度、抗拉强度、疲劳寿命 — 车身钢材/铝合金在碰撞安全中的角色 |
| 4 | **工程材料概述** | 钢（高强钢/热成型钢）、铝合金、镁合金、碳纤维、工程塑料在汽车上的分布（白车身/覆盖件/内饰） |
| 5 | **热力学基础** | 热功转换、卡诺循环、热效率 — 为理解发动机（奥托循环/阿特金森循环）和空调制冷做准备 |
| 6 | **传动基础** | 扭矩/转速/功率关系（P=τ·ω），齿轮传动比、减速增扭原理，为变速箱学习打底 |
| 7 | **流体力学初步** | 空气阻力公式（F=½ρCdAv²）、风阻系数 Cd、气动升力 — 为什么跑车要低趴 |
| 8 | **振动与噪声入门** | 频率/振幅/共振，NVH（Noise, Vibration, Harshness）的基本概念 |

### 前置依赖

- **第1层 整车认知**：需要先对汽车整体有感性认识，才能理解力学模型在车上的对应位置

### 学习目标

学完本层后，学员能够：
1. 用牛顿定律解释汽车加速、制动和转弯时的受力变化
2. 说出白车身主要用了哪几种钢材，以及各自承担的功能
3. 解释为什么变速箱需要「减速增扭」
4. 看懂风阻系数数值，判断一辆车的气动设计水平

## 第3层：传统系统

> **定位**：拆解传统燃油车的六大核心系统，理解每个子系统的工作原理和常见结构。
> **预计学时**：12–16 小时（本层信息量最大，建议分两段学）

### 关键知识点

| # | 知识点 | 简介 |
|---|--------|------|
| 1 | **发动机工作原理** | 四冲程（进气-压缩-做功-排气）、奥托循环 vs 阿特金森循环、直列/V型/水平对置布局；自然吸气/涡轮增压/机械增压的区别 |
| 2 | **发动机关键参数** | 排量、压缩比、最大功率/最大扭矩及其转速区间、升功率、热效率 — 「看发动机曲线图」 |
| 3 | **变速箱** | MT（手动）、AT（液力自动）、CVT（无级变速）、DCT/DSG（双离合）— 各自结构与优缺点 |
| 4 | **底盘系统总览** | 底盘 = 传动系统 + 行驶系统 + 转向系统 + 制动系统（四大子系统切分） |
| 5 | **悬架系统** | 独立悬架（麦弗逊/多连杆/双叉臂）vs 非独立悬架（扭力梁）；弹簧+减振器的作用；悬架调校「软硬」的含义 |
| 6 | **转向系统** | 齿轮齿条式、助力转向（液压 HPS / 电动 EPS）、转向比、转弯半径 |
| 7 | **制动系统** | 盘式/鼓式制动、制动助力（真空/电子）、ABS（防抱死）、ESC（电子稳定控制）、制动能量回收（为第4层铺垫） |
| 8 | **汽车电气系统** | 12V 电路、蓄电池/发电机、线束、CAN 总线基础（为第5层域控制器铺垫）；车灯、雨刮、中控锁等车身电子 |
| 9 | **排放与后处理** | 三元催化器、GPF（汽油颗粒捕捉器）、国六b 标准 — 为什么排气管里有「口罩」 |
| 10 | **空调与热管理** | 制冷循环（四部件：压缩机/冷凝器/膨胀阀/蒸发器）、热泵制热（为第4层电池热管理铺垫） |

### 前置依赖

- **第1层 整车认知**：需要知道发动机/底盘/电气在整车中的位置和角色
- **第2层 机械基础**：发动机工作原理依赖热力学（卡诺循环/奥托循环）；底盘各系统依赖力学/摩擦/材料/传动基础

### 学习目标

学完本层后，学员能够：
1. 画出四冲程发动机的工作循环图，并区分自然吸气和涡轮增压的工作差异
2. 面对一款车的参数表，解释其变速箱类型（AT/CVT/DCT）并给出选购建议
3. 说出麦弗逊 vs 双叉臂悬架的典型应用场景（买菜车 vs 性能车）
4. 解释 ABS/ESC 分别「防什么」、怎么工作

## 第4层：新能源

> **定位**：聚焦电动汽车核心技术（三电系统），覆盖混动、充电基础设施和热管理。
> **预计学时**：10–12 小时

### 关键知识点

| # | 知识点 | 简介 |
|---|--------|------|
| 1 | **动力电池基础** | 电芯/模组/PACK 三级结构；锂离子电池工作原理（Li⁺ 在正负极间移动）；三元锂（NCM/NCA）vs 磷酸铁锂（LFP）— 能量密度/安全/成本三角 |
| 2 | **电池关键指标** | 标称电压、容量（Ah）、能量（kWh）、能量密度（Wh/kg, Wh/L）、SOC（荷电状态）、SOH（健康状态）、循环寿命、C 倍率 |
| 3 | **驱动电机** | 永磁同步电机（PMSM）vs 交流异步电机（感应电机）；电机特性曲线（恒扭矩区/恒功率区）；扁线/油冷等新技术 |
| 4 | **电控系统（MCU/VCU/BMS）** | MCU（电机控制器）：逆变/整流；VCU（整车控制器）：扭矩分配、驾驶模式；BMS（电池管理系统）：SOC 估算、均衡、热管理 |
| 5 | **混动架构** | 串联（增程式，REEV）、并联、混联（功率分流）；P0–P4 电机位置；比亚迪 DM-i、理想增程、丰田 THS 案例对比 |
| 6 | **充电系统** | 交流慢充（AC，家用桩 7kW）vs 直流快充（DC，超充 250kW+）；充电接口标准（国标 GB/T、CCS、NACS）；充电曲线与「补能焦虑」 |
| 7 | **电池热管理** | 风冷/液冷/直冷；低温对电池的影响（续航缩水、充电变慢）；热泵空调在冬季节能中的作用 |
| 8 | **高压电气架构** | 400V → 800V 平台演进；SiC（碳化硅）功率器件的优势；为什么 800V 能「充电 5 分钟续航 200 公里」 |
| 9 | **燃料电池（FCEV）** | 氢燃料电池工作原理（质子交换膜）、储氢方式、加氢站 — 现状与技术挑战 |

### 前置依赖

- **第2层 机械基础**：电机特性曲线依赖扭矩/转速/功率关系；电池热管理依赖热力学基础
- **第3层 传统系统**：电控替代了传统燃油车的许多机械部件，需要对比理解；制动能量回收并联第3层制动系统；电气系统基础关联第3层汽车电气

### 学习目标

学完本层后，学员能够：
1. 独立分辨三元锂电池和磷酸铁锂电池，并说出各自适合什么车型/场景
2. 解释一台纯电车的「三电」分别是什么，各自干什么
3. 对比一台增程式混动和一台插电混动的动力流走向
4. 看懂充电功率和充电曲线，估算一辆车从 30% 充到 80% 需要多长时间

## 第5层：智能汽车

> **定位**：理解汽车从「交通工具」到「移动智能终端」的技术跃迁，覆盖智能座舱、辅助驾驶、电子电气架构。
> **预计学时**：10–12 小时

### 关键知识点

| # | 知识点 | 简介 |
|---|--------|------|
| 1 | **电子电气架构演进** | 分布式 ECU → 域集中（五域：动力域/底盘域/座舱域/智驾域/车身域）→ 中央计算+区域控制器；为什么需要「减少 ECU 数量」 |
| 2 | **域控制器（Domain Controller）** | 座舱域控制器 vs 智驾域控制器；高算力 SoC（高通 8295/英伟达 Orin/地平线征程）；「行泊一体」方案 |
| 3 | **智能座舱** | 车机 OS（Android Automotive / 鸿蒙座舱 / 蔚来 NIO OS）；多屏交互与 HUD；语音助手与多模态交互；OTA 升级能力 |
| 4 | **辅助驾驶分级（ADAS）** | L0–L5（SAE J3016 分级标准）；中国国标 GB/T 40429 对应分级；从 AEB/ACC/LKA 到 NOA（高速/城市领航辅助） |
| 5 | **感知系统** | 摄像头（单目/双目/环视）、毫米波雷达、超声波雷达、激光雷达（LiDAR）— 各自原理、优劣势和典型上车方案 |
| 6 | **高精定位与地图** | GNSS/RTK/IMU 组合定位；高精地图与 SD Map、BEV（鸟瞰视图）感知、无图化趋势 |
| 7 | **决策规划与控制** | 感知→预测→规划→控制的流水线；行为决策（FSM/RL）vs 端到端（模仿学习/世界模型）；「影子模式」 |
| 8 | **软件定义汽车（SDV）** | SOA 服务化架构、车载以太网、SOME/IP 协议；为什么车企开始「自研操作系统」；特斯拉/蔚小理的软件迭代案例 |
| 9 | **安全与功能安全** | ISO 26262 功能安全（ASIL 等级）、ISO 21434 网络安全、SOTIF（预期功能安全）— 智能驾驶的安全底线 |
| 10 | **V2X 车联网** | V2V（车-车）/ V2I（车-路）/ V2P（车-人）/ V2N（车-云）；C-V2X（中国主推）vs DSRC；「车路云一体化」国家战略 |

### 前置依赖

- **第2层 机械基础**：感知系统涉及电磁波/光学基础；域控制器涉及计算原理
- **第3层 传统系统**：需要先理解传统汽车的「分散式」电子电气架构，才能理解「域集中」的价值；制动/转向系统是执行层基础（线控制动/线控转向）
- **第4层 新能源**（部分依赖）：三电系统通过整车控制器（VCU）与智能驾驶域联动；高电压平台为高算力域控供电

### 学习目标

学完本层后，学员能够：
1. 画出传统分布式 ECU 架构 → 域集中架构 → 中央计算架构的演进图
2. 复述 L0–L5 的辅助驾驶分级标准，并举例每一级别对应的量产功能
3. 对比摄像头、毫米波雷达、激光雷达各自的感知能力和短板
4. 解释「软件定义汽车」为什么是行业趋势，并举例 OTA 升级如何改变用车体验

## 第6层：车企工作语境

> **定位**：把技术知识嵌入车企真实的研发流程和日常工作中，让学员「不光懂技术，还能说黑话」。
> **预计学时**：6–8 小时

### 关键知识点

| # | 知识点 | 简介 |
|---|--------|------|
| 1 | **整车开发流程（V-Cycle）** | 概念设计 → 详细设计 → 零部件验证 → 系统集成 → 整车验证 → SOP（批量投产）；每个阶段做什么、产出什么 |
| 2 | **研发节点与里程碑** | 项目立项/概念批准/G8-G0 阀点体系（各车企叫法不同但逻辑相通）：骡子车（Mule Car）、EP/OTS/PPAP/SOP 节点含义 |
| 3 | **车企岗位分工** | 产品规划/造型设计/工程开发（动力/底盘/车身/电子电器/NVH）/试验验证/工艺/质量/采购/项目管理 — 30 秒说清每个岗位干什么 |
| 4 | **行业黑话词典** | 对标（Benchmarking）、平台化/模块化、减震器/避震器、点头/侧倾/推头/甩尾、热成型/冷冲压、黑车身/白车身、爬产/拉量、小改款/中期改款/全新换代 |
| 5 | **供应链体系** | Tier1/Tier2/Tier3 分级；一级供应商巨头（博世/大陆/电装）；中国本土供应链崛起（宁德时代/地平线/禾赛/德赛西威等） |
| 6 | **法规认证与检测** | 公告认证（工信部公告）、CCC 认证、国六排放、C-NCAP/E-NCAP 碰撞安全评级、三包法 |
| 7 | **生产制造入门** | 冲/焊/涂/总 四大工艺；节拍（JPH, Jobs Per Hour）；柔性生产线 |
| 8 | **项目管理概念** | APQP（先期产品质量策划）、DFMEA（设计失效模式分析）、8D 问题解决法 — 车企通用的质量/项目工具 |
| 9 | **行业媒体与信息源** | 汽车之家/懂车帝/易车评测体系；SAE 论文；工信部公告；各车企财报发布会关键看点 |

### 前置依赖

- **第1层–第5层全部**：需要先理解各系统的技术内容，才能理解为什么研发流程要按这个顺序走、为什么某个岗位需要某个技能、黑话的底层技术含义

### 学习目标

学完本层后，学员能够：
1. 画出一张整车开发 V 形图，标注各阶段的关键节点和产出物
2. 在聊天中听到「这车还在 EP 阶段」时，准确理解项目进度
3. 说出至少 5 个 Tier1 供应商及其主营业务领域
4. 描述一款车从「领导说要做」到「4S 店开卖」的全流程周期
