# 旗舰交互课

> 6 门旗舰交互课，配合 30 天学习地图，用「场景→图→原理→对比→工作→小测」的结构帮助你深入理解核心概念。

## 课程列表

| # | 课程 | 主题 |
|---|------|------|
| 1 | [一辆车由哪些系统组成](./01-car-systems) | 六大系统全局图 |
| 2 | [燃油车如何把燃油变成轮端驱动力](./02-fuel-to-wheel) | 动力链全路径 |
| 3 | [踩刹车时力/液压如何传递](./03-braking-force) | 制动系统原理 |
| 4 | [电动车高压能量流](./04-ev-energy-flow) | 三电系统与能量管理 |
| 5 | [智能驾驶一次感知-决策-执行闭环](./05-adas-loop) | ADAS 技术栈 |
| 6 | [新人第一次参加问题分析会如何听懂术语](./06-newcomer-terminology) | 车企工作语境 |

::: tip 学习建议
每门课约 15-20 分钟，建议配合 [30 天学习地图](/path) 的节奏完成。课程 1-4 对应前两周，课程 5-6 对应第四周。
:::

## 互动功能说明

旗舰课支持以下交互功能：

- **术语卡**：遇到缩写术语（如 **BMS**、**ECU**），悬停查看释义（参见 [术语速查表](/glossary)）
- **小测**：每门课末尾有选择题，即时反馈
- **进度记录**：勾选「标记完成」记录学习进度
