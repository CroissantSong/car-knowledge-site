## 行业媒体与常用信息源

新人进入车企后，需要快速建立信息获取习惯。以下是按用途分类的常用信息源清单：

### 一、中文汽车媒体（行业动态 / 新车 / 评测）

| 媒体 | 类型 | 特色 | 适合关注点 |
|------|------|------|-----------|
| **汽车之家** | 综合门户 | 车型库最全、参数对比、车主口碑 | 竞品参数、用户反馈 |
| **懂车帝** | 综合门户 | 视频评测为主、冬测/夏测数据 | 实测续航、AEB 横评 |
| **易车** | 综合门户 | 经销商报价、购车计算器 | 终端价格、优惠行情 |
| **42 号车库** | 深度媒体 | 智能驾驶评测、技术解析 | 智驾/座舱/电子电气 |
| **GeekCar 极客汽车** | 深度媒体 | 智能汽车技术、行业访谈 | 智能座舱、车联网 |
| **XEV 研究所** | 技术自媒体 | 电池/电驱/EE 架构技术拆解 | 三电系统、高压架构 |

### 二、行业数据与政策

| 来源 | 网址/说明 | 用途 |
|------|----------|------|
| **工信部公告** | miit.gov.cn，每月发布《道路机动车辆生产企业及产品公告》 | 新车上市前必须查的合规信息 |
| **乘联会** | cpcaauto.com，月度产销数据和市场分析 | 销量排名、细分市场趋势 |
| **中国汽车工业协会** | caam.org.cn，年度/月度行业统计 | 行业总量、进出口数据 |
| **电池联盟** | 动力电池月度装机量排名 | 电池厂份额、技术路线趋势 |
| **新能源汽车国家监测平台** | 实时接入车辆运行数据 | 补贴核查、安全监管 |

### 三、国际标准与法规

| 来源 | 说明 |
|------|------|
| **SAE International** | sae.org，SAE J3016 自动驾驶分级、各类技术标准 |
| **UNECE WP.29** | 联合国车辆法规协调论坛，R155 网络安全、R156 软件更新 |
| **NHTSA** | 美国国家公路交通安全管理局，碰撞测试、召回 |
| **Euro NCAP** | 欧洲新车安全评鉴，碰撞 & 主动安全评分 |
| **C-NCAP / C-IASI** | 中国新车评价规程 / 中国保险汽车安全指数 |

### 四、企业财报与一手信息

| 来源 | 用途 |
|------|------|
| **各车企官网投资者关系页** | 季度/年度财报、交付量、技术路线图 |
| **36氪 / 晚点 Auto / 虎嗅汽车组** | 国内造车新势力融资、组织变动、供应链动态 |
| **CleanTechnica / InsideEVs** | 全球电动车销量追踪、海外市场分析 |
| **宁德时代 / 比亚迪 / 地平线 / 禾赛 官方公众号** | 供应商技术发布会、新品参数第一手 |

### 五、新人推荐关注组合

如果精力有限，建议先盯住以下 **5 个信息源**：

1. **乘联会月度销量** — 知道谁在涨、谁在跌（每周一看）
2. **工信部公告** — 知道谁要发新车（每月一批）
3. **懂车帝实测** — 知道实际续航/加速/制动距离
4. **42 号车库** — 知道智驾/座舱技术趋势
5. **公司内部分享会** — 最好的信息来源其实是同事

**小测**：如果你想在某款新车上市前就知道它的电池容量和电机功率，应该去查哪个来源？
