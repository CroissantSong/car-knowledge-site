# 扭矩与马力

> 这是汽车领域最容易混淆的概念之一。本文帮助非汽车专业的你一次性搞懂。

## 一句话总结

| 概念 | 通俗理解 | 对应感受 |
|------|----------|----------|
| **扭矩** | 「力气」——发动机一次做功能输出多大的旋转力 | 起步/超车时的「推背感」 |
| **马力** | 「功率」——单位时间内能做多少功 | 最高速度能力和持续加速能力 |

> **扭矩决定加速有多猛，马力决定能跑多快。**

## 物理定义

### 扭矩（Torque）

$$
\tau = F \times r
$$

- $\tau$：扭矩（N·m，牛·米）
- $F$：作用力
- $r$：力臂长度

发动机的扭矩是曲轴输出的旋转力矩。可以用扳手拧螺丝来类比：

> 你用 1m 长的扳手施加 200N 的力 = 200 N·m 的扭矩。

### 马力（Horsepower / Power）

$$
P = \tau \times \omega = \frac{\tau \times \text{RPM}}{9550}
$$

- $P$：功率（kW）
- $\tau$：扭矩（N·m）
- $\omega$ / RPM：转速

### 扭矩与马力的关系

> **马力 = 扭矩 × 转速 ÷ 9550**（单位：kW、N·m、rpm）

同一台发动机：

- 低转速 → 扭矩可能不错但马力不大（因为转速低）
- 高转速 → 即使扭矩下降，马力也可能上升（转速高弥补）

这就是为什么发动机有「扭矩曲线」和「功率曲线」两条线：

```
扭矩 ↑                    功率 ↑
     │    ╭──╮                 │      ╱────╲
     │   ╱    ╲___             │    ╱        ╲___
     │  ╱                             │  ╱
     │ ╱                              │ ╱
     │╱                               │╱
     └──────────→ RPM        └──────────→ RPM
```

## 实际驾驶中的感受

### 柴油机 vs 汽油机

| 类型 | 扭矩 | 马力 | 驾感 |
|------|------|------|------|
| **柴油机** | 低转大扭矩 | 高转无力 | 起步猛，但转速上不去 |
| **汽油机** | 扭矩曲线更平 | 高转马力大 | 转速拉高后有劲 |
| **涡轮汽油机** | 低转就有大扭矩（涡轮介入后） | 平台期马力 | 中低转就很猛 |

### 电动车为什么快？

电机从 0 转速开始就能输出**峰值扭矩**，不需要像内燃机那样等待转速爬升。

```
内燃机：扭矩随转速逐渐上升 → 起步慢
电机：  0rpm 就输出最大扭矩 → 起步快
```

这就是为什么特斯拉、比亚迪汉等电动车起步极快——电机物理特性使然。

## 看参数时要注意什么

### 峰值数据 ≠ 实际表现

两台发动机可能都标称「200PS / 300N·m」，但驾驶感受可能完全不同：

- **扭矩平台**：涡轮机可能 1500-4000rpm 都是峰值扭矩，而自吸机只在某个点达到
- **低扭**：1000-2000rpm 的扭矩决定日常市区驾驶的轻快感

### 常用判断

| 你问 | 看哪个 |
|------|--------|
| 这车超车快不快？ | 扭矩 + 扭矩平台宽度 |
| 这车最高能跑多快？ | 最大马力 |
| 这车城市里好开吗？ | 低扭（1500rpm左右的扭矩输出） |
| 这车拉货/爬坡行不行？ | 扭矩 + 低速扭矩 |

## QA：常见疑问

**Q：扭矩大就一定快吗？**  
A：不完全。还要看变速箱齿比、车重、轮胎抓地力等。扭矩是发动机输出的「潜力」，实际表现取决于传动系统的匹配。

**Q：「马力越大越快」对吗？**  
A：理论上对。但日常驾驶很少用到最大马力——你更多时候感受到的是 2000-4000rpm 区间的扭矩表现。

**Q：自然吸气和涡轮增压怎么选？**  
A：自吸输出线性、响应快（踩多少给多少）；涡轮中低转更有力但可能有延迟感（涡轮迟滞）。如今涡轮技术已经很成熟，迟滞越来越小。
