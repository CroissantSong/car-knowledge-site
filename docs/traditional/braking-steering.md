# 制动与转向

## 制动系统

### 制动原理

将车辆的动能转化为热能，使车辆减速或停止。

> 摩擦力 = 正压力 × 摩擦系数

### 分类

| 类型 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **盘式制动** | 卡钳夹紧制动盘 | 散热好、抗热衰退 | 成本较高 |
| **鼓式制动** | 制动蹄撑开压紧制动鼓 | 成本低、制动力大 | 散热差、易衰退 |

现代乘用车普遍采用**前盘后盘**或**前盘后鼓**配置。

### 制动系统组成

```
制动踏板 → 真空助力器 → 制动总泵 → 制动液 → 制动分泵 → 制动片/蹄 → 制动盘/鼓
```

- **制动液（刹车油）**：DOT3/DOT4/DOT5.1，需定期更换（吸湿后沸点降低）
- **真空助力器**：利用发动机进气歧管真空度放大踏板力

## ABS（防抱死制动系统）

### 为什么要 ABS？

无 ABS 紧急制动时，车轮抱死 → 失去转向能力 + 制动距离可能更长（滑动摩擦＜最大静摩擦）。

### ABS 工作原理

1. 轮速传感器监测各轮转速
2. 发现某轮即将抱死 → ABS 控制单元调节制动压力
3. 快速「点刹」（每秒约 15-30 次），保持车轮在滚动与滑动的临界状态
4. 最大化制动力的同时保留转向能力

## ESP（电子稳定程序）

防止车辆侧滑失控。

### 工作场景

- **转向不足（推头）**：制动内侧后轮，将车头拉回弯内
- **转向过度（甩尾）**：制动外侧前轮，稳定车尾

ESP 集成了 ABS、TCS（牵引力控制）等功能。

## 转向系统

### 类型演变

| 类型 | 助力来源 | 路感 | 能耗 |
|------|----------|------|------|
| **机械转向** | 人力 | 直接 | 0 |
| **液压助力（HPS）** | 发动机驱动油泵 | 较好 | 持续耗能 |
| **电动助力（EPS）** | 电机 | 可调 | 仅转向时耗能 |

目前主流是 **EPS**（电动助力），其优势：

- 节能（不转向时不耗能）
- 可编程（低速轻便、高速稳重）
- 支持 ADAS（车道保持、自动泊车等需要主动转向）

### 转向参数

- **转向比**：方向盘转角度与车轮转角度的比值
  - 一般乘用车：14:1 ~ 18:1
  - 运动型车更小（更直接），大体型车更大
- **最小转弯半径**：影响窄路掉头能力

## 轮胎知识

### 规格解读

以 `205/55 R16 91V` 为例：

```
205 / 55  R16  91 V
 │     │    │   │  └─ 速度等级（V=240km/h）
 │     │    │   └── 载重指数（91=615kg）
 │     │    └────── 轮辋直径（16英寸）
 │     └─────────── 扁平比（胎高/胎宽=55%）
 └───────────────── 胎面宽度（205mm）
```

### 常见速度等级

| 代号 | 最高速度 | 代号 | 最高速度 |
|------|----------|------|----------|
| S | 180 km/h | V | 240 km/h |
| T | 190 km/h | W | 270 km/h |
| H | 210 km/h | Y | 300 km/h |
