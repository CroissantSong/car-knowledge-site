# 燃油动力系统

## 燃油系统

将燃油从油箱输送到发动机气缸的过程。

```
油箱 → 燃油泵 → 燃油滤清器 → 燃油导轨 → 喷油嘴 → 气缸
```

### 各部件说明

| 部件 | 功能 |
|------|------|
| **油箱** | 储存燃油，内置隔板防晃 |
| **燃油泵** | 将燃油加压输送（通常浸在油箱内） |
| **燃油滤清器** | 过滤杂质 | 
| **喷油嘴** | 将燃油雾化喷入进气歧管或气缸（直喷） |

## 进气系统

为发动机提供清洁空气。

```
空气 → 空气滤清器 → 空气流量计 → 节气门 → 进气歧管 → 气缸
```

- **空气滤清器（空滤）**：过滤空气中的灰尘和颗粒
- **节气门（Throttle）**：控制进入发动机的空气量，由油门踏板控制
- **进气歧管**：将空气均匀分配到各气缸

## 排气系统

将燃烧后的废气排出，降低噪音和排放。

```
气缸 → 排气歧管 → 三元催化器 → 消声器 → 尾管
```

- **排气歧管**：收集各气缸废气
- **三元催化器（TWC）**：将 CO、HC、NOx 转化为 CO₂、H₂O、N₂
- **消声器**：降低排气噪音

## 涡轮增压（Turbocharger）

### 原理

用废气能量驱动涡轮，带动压气机压缩进气，在相同排量下泵入更多空气，从而喷更多油、做更多功。

```
废气 → 涡轮（转动）→ 同轴 → 压气机（转动）→ 压缩进气 → 中冷器 → 进入气缸
```

### 为什么涡轮增压是趋势？

- **排量小型化**：1.5T 可输出接近 2.5L 自然吸气的功率
- **降低税费**：中国按排量征收购置税和车船税
- **节油**：低负荷时如小排量自吸，高负荷时增压补强

### 注意

- **涡轮迟滞（Turbo Lag）**：踩下油门到涡轮起压之间的延迟感
- 缓解方法：小惯量涡轮、双涡管、可变截面涡轮（VGT）

## 冷却系统

维持发动机在最佳温度范围（85-95℃）。

```
┌─────────────────────────────────────┐
│     发动机                             │
│  ┌─────────┐      ┌─────────┐        │
│  │ 水套     │ ←── │ 水泵     │        │
│  └────┬────┘      └────┬────┘        │
│       │                │              │
│       ↓                ↑              │
│  ┌─────────┐      ┌─────────┐        │
│  │ 节温器   │ ──→  │ 散热器   │        │
│  └─────────┘      └─────────┘        │
└─────────────────────────────────────┘
```

- **冷却液**：乙二醇水溶液（防冻防沸），不能用水替代
- **节温器**：控制冷却液循环路径（冷机时小循环暖机，热机时大循环散热）
- **散热风扇**：低速/堵车时强制散热

## 润滑系统

减少摩擦、冷却、清洁、防锈、密封。

- **机油泵**：将机油加压输送到各摩擦副
- **机油滤清器（机滤）**：过滤金属碎屑和积碳
- **机油标号**：如 5W-30
  - **W 前数字**：低温流动性（越小越耐低温）
  - **W 后数字**：高温粘度（越大越稠）
