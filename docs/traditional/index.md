# 第三层：传统系统 🔧

《深入汽车的各个功能子系统》

前两层建立了整车和核心机械的认知。本层深入到燃油车的各个功能子系统——它们怎样协同工作让一辆车安全、可靠地行驶。

## 本层知识点

| # | 知识点 | 说明 |
|---|--------|------|
| 19 | 燃油系统构成 | 油箱 → 油泵 → 滤清器 → 喷油嘴 |
| 20 | 进气与排气系统 | 空滤 / 节气门 / 进气歧管 / 排气歧管 / 消声器 |
| 21 | 涡轮增压原理 | 用废气能量压缩进气，提升功率密度 |
| 22 | 冷却系统 | 散热器 / 水泵 / 节温器 / 冷却液循环 |
| 23 | 润滑系统 | 机油泵 / 机油滤清器 / 油道 / 油底壳 |
| 24 | 制动系统类型 | 盘式制动 vs 鼓式制动 |
| 25 | ABS / ESP | 防抱死制动 / 电子稳定程序的工作原理 |
| 26 | 转向系统 | 齿轮齿条 / 液压助力 / 电动助力 |
| 27 | 轮胎知识 | 规格解读（如 205/55 R16 91V）/ 胎压 / 磨损 |
