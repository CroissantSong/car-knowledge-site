# 📖 术语速查表

> 汽车行业常用缩写与术语，悬停可查看释义。这些术语在日常会议、文档和工作中高频出现。

## 三电系统

| 缩写 | 全称 | 释义 |
|------|------|------|
| **BMS** | Battery Management System | 电池管理系统，监控电芯状态 |
| **MCU** | Motor Control Unit | 电机控制器，含逆变器 |
| **VCU** | Vehicle Control Unit | 整车控制器，协调扭矩分配 |
| **SOC** | State of Charge | 荷电状态（电量百分比） |
| **SOH** | State of Health | 健康状态（容量保持率） |
| **PMSM** | Permanent Magnet Synchronous Motor | 永磁同步电机 |
| **SiC** | Silicon Carbide | 碳化硅，第三代半导体材料 |
