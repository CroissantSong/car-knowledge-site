## 🚗 按六层练习

### 第一层：整车认知

建立整车系统全局观——六大系统、白车身、VIN码、分类标准。

[开始练习 →](./guide)

### 第二层：机械基础

扭矩与马力、差速器原理、四冲程循环——连接驾驶动作与机械原理。

[开始练习 →](./mechanics)

### 第三层：传统系统

燃油动力链效率、制动系统液压原理、制动液安全——从踩油门/刹车理解车辆动作。

[开始练习 →](./traditional)

### 第四层：新能源

三电系统、充电对比、电池安全、800V高压、增程vs插混——理解电动车与传统车的本质差异。

[开始练习 →](./new-energy)

### 第五层：智能汽车

感知-决策-执行闭环、线控制动、传感器方案、自动驾驶分级——理解"软件定义汽车"。

[开始练习 →](./smart-car)

### 第六层：车企工作语境

NVH/SOP/FMEA/8D/Tier——掌握车企会议中的工程语言。

[开始练习 →](./industry)

## 📦 数据说明

题库数据文件：`docs/.vitepress/data/quiz/bank.js`

题目结构：

```js
{
  id: '唯一标识',
  type: 'single' | 'multi' | 'judge',  // 单选 | 多选 | 判断
  question: '题目文本',
  options: [{ text: '选项', correct: true/false }],
  explanation: '正确答案解析',
  layer: 'guide',     // 六层归属
  role: 'all',        // 岗位关注
  tags: ['标签1', '标签2']
}
```

::: info 公式与术语
题目中涉及数学公式时使用 `$...$` 包装（如 `$P = V \\times I$`），
术语使用 `<TermCard>` 组件确保悬停查看释义。
:::
