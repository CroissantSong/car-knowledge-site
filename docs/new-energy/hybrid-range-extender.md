# 混合动力与增程

## 混合动力分类

混合动力汽车（HEV/PHEV）同时搭载内燃机和电动机。

### 按电动化程度

| 类型 | 电池容量 | 纯电续航 | 充电方式 | 代表 |
|------|----------|----------|----------|------|
| **轻混（MHEV）** | 极小（48V） | 不能纯电行驶 | 能量回收 | 多数新车标配 |
| **全混（HEV）** | 1-2 kWh | 几公里 | 能量回收 | 丰田 THS、本田 i-MMD |
| **插混（PHEV）** | 10-40 kWh | 50-200 km | 可外部充电 | 比亚迪 DM-i |

### 按结构分类

```
串联 (Series Hybrid)
┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐
│发动机│→│发电机│→│ 电池 │→│ 电机 │→ 车轮
└─────┘   └─────┘   └─────┘   └─────┘
发动机不直接驱动车轮，仅用于发电（增程式）

并联 (Parallel Hybrid)
┌─────┐   ┌─────┐
│发动机│→  │变速箱│──→ 车轮
└─────┘   └──┬──┘
┌─────┐      │
│ 电机 │─────┘
└──┬──┘
   └── 电池
发动机和电机都可以直接驱动车轮

混联 (Power-Split Hybrid)
┌─────┐   ┌─────┐
│发动机│→  │行星  │──→ 车轮
└─────┘   │齿轮  │
┌─────┐   │组    │
│发电机│←→│      │←→ 电机
└─────┘   └─────┘
通过行星齿轮组实现功率分流（丰田THS经典方案）
```

### 各方案对比

| 方案 | 代表 | 优点 | 缺点 |
|------|------|------|------|
| 串联（增程） | 理想 ONE/L 系列 | 结构简单、发动机始终高效运行 | 高速油耗较高 |
| 并联 | 早期方案 | 两套动力叠加 | 控制复杂，已少见 |
| 混联（功率分流） | 丰田 THS | 全工况高效 | 机制复杂 |

## 增程式电动车（EREV）

发动机不参与驱动，仅作为「发电机」。

### 工作原理

1. 电池有电：纯电行驶
2. 电池电量低：发动机启动 → 发电 → 供电机驱动 + 给电池充电
3. 高速巡航时，发动机可直接在高效区间运行

### 为什么增程受欢迎？

- **无里程焦虑**：加油即可继续行驶
- **大部分时间纯电**：日常通勤用电，长途用油
- **技术简单可靠**：无需复杂的混动变速箱

### 争议

> 「增程是过渡技术还是长期路线？」

- **支持**：电池成本下降后，增程器可拆除变为纯电；发动机效率提升空间小
- **反对**：多了一套系统，高速效率不如高效混联

## 充电技术

### 慢充（AC）

| 参数 | 值 |
|------|-----|
| 电压 | 220V 单相（中国）/ 380V 三相 |
| 电流 | 16A / 32A |
| 功率 | 3.3kW / 7kW |
| 充满时间 | 6-12 小时 |

### 快充（DC）

| 参数 | 值 |
|------|-----|
| 电压 | 400-800V |
| 功率 | 60kW / 120kW / 250kW / 480kW+ |
| 30%-80% 时间 | 约 20-30 分钟 |

### 800V 高压平台

充电功率 P = V × I，提高电压可降低电流（减少发热和损耗）。

> 800V 平台可将 10%-80% 充电时间压缩到 15 分钟以内。

### 其他充电方式

- **换电**：直接更换电池包（蔚来等），3-5 分钟完成
- **无线充电**：停车位感应充电，功率较低（7-11kW），尚未普及

## 能量回收

制动时电机反转作发电机，将动能转化为电能充入电池。

- **回收效率**：通常可回收 15-30% 的能量
