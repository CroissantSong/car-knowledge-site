# 电池与电机

## 纯电动汽车架构

纯电动汽车（BEV）的核心是「三电系统」：

```
┌────────────────────────────────────────┐
│                整车控制器 (VCU)           │
│                                          │
│  ┌──────────┐  ┌──────────┐  ┌───────┐  │
│  │ 动力电池   │  │电机控制器 │  │ 电机   │  │
│  │ + BMS    │→ │  MCU     │→ │ Motor │──┼──→ 车轮
│  └──────────┘  └──────────┘  └───────┘  │
│       ↑                        ↑         │
│  充电接口                   能量回收      │
└────────────────────────────────────────┘
```

## 动力电池

### 主流类型对比

| 特性 | 三元锂 NCM/NCA | 磷酸铁锂 LFP | 固态电池（未来） |
|------|---------------|-------------|----------------|
| 能量密度 | 高（200-260 Wh/kg） | 中（140-180 Wh/kg） | 极高（400+ Wh/kg） |
| 安全性 | 热失控温度低 | 热失控温度高，更安全 | 本质安全 |
| 循环寿命 | 800-1500 次 | 2000-4000 次 | 长 |
| 成本 | 较高（含钴/镍） | 低（无贵金属） | 目前极高 |
| 低温性能 | 较好 | 较差 | 待验证 |

### 电池包结构

```
电芯 (Cell) → 模组 (Module) → 电池包 (Pack)
```

- **电芯**：最小单元，常见型式为方形、圆柱、软包
- **模组**：多个电芯串联/并联组成的单元
- **电池包**：多个模组 + BMS + 热管理系统 + 外壳

### 电池参数

- **容量**：单位 kWh（度），如 80kWh 电池包
- **电压平台**：通常 300-400V，800V 高压平台是趋势（充电更快）
- **SOC（State of Charge）**：剩余电量百分比，0-100%

## 电池管理系统 BMS

BMS 是电池包的「大脑」，负责：

| 功能 | 说明 |
|------|------|
| **SOC 估算** | 估计剩余电量（不是简单的电压对应） |
| **均衡管理** | 消除各电芯间的电压/容量差异 |
| **热管理** | 加热（低温时）和散热（高功率时） |
| **安全保护** | 过充/过放/过温/短路保护 |

## 驱动电机

### 主流类型

| 类型 | 特点 | 应用 |
|------|------|------|
| **永磁同步电机（PMSM）** | 高效率、高功率密度、需稀土 | 大多数国产电动车 |
| **异步感应电机（IM）** | 无需稀土、高转速、效率稍低 | 特斯拉前电机（部分车型） |

### 电机关键参数

- **峰值功率**：如 200kW，决定加速能力
- **峰值扭矩**：电机从 0 转速即可输出峰值扭矩（这是电动车起步快的原因）
- **最高转速**：通常 12000-20000 rpm，影响极速

## 电机控制器 MCU

将电池的直流电（DC）逆变为驱动电机的交流电（AC），并控制电机的转速和扭矩。

```
电池 DC → MCU（DC-AC 逆变）→ 变频 AC → 电机
```

核心技术：
- **IGBT**：主流的功率半导体器件
- **SiC（碳化硅）**：新一代材料，更低损耗、更高频率，正在替代 IGBT
