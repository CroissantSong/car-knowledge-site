# 汽车分类与结构

## 汽车的定义

根据中国国家标准 GB/T 3730.1-2001，汽车（automobile）是指由动力驱动，具有四个或四个以上车轮的非轨道承载车辆，主要用于载运人员和/或货物。

## 分类体系

### 按用途分类

| 类别 | 说明 | 示例 |
|------|------|------|
| **乘用车** | 主要用于载运乘客及其随身行李 | 轿车、SUV、MPV |
| **商用车** | 主要用于载运货物或人员（营运） | 货车、客车、牵引车 |
| **特种车** | 用于特殊用途 | 消防车、救护车、工程车 |

### 按车身形式分类（乘用车）

| 形式 | 特点 | 代表车型 |
|------|------|----------|
| **三厢车（Sedan）** | 发动机舱+乘员舱+行李舱，三厢分明 | 大众迈腾、丰田凯美瑞 |
| **两厢车（Hatchback）** | 乘员舱与行李舱连通 | 大众高尔夫、本田飞度 |
| **SUV** | 运动型多用途车，高底盘、大空间 | 哈弗H6、特斯拉Model Y |
| **MPV** | 多功能乘用车，7座布局 | 别克GL8、丰田赛那 |
| **跑车（Sport Car）** | 低重心、强调性能 | 保时捷911、小米SU7 Ultra |

### 按动力类型分类

- **燃油车（ICE）**：以内燃机为唯一动力源
- **混合动力（HEV/PHEV）**：内燃机+电动机
- **纯电动（BEV）**：仅电池+电机驱动
- **燃料电池（FCEV）**：氢燃料电池发电

## 车辆识别代号 VIN

VIN（Vehicle Identification Number）是车辆的全球唯一标识，由 17 位字符组成：

```
L F V 3 A 2 1 K 1 M 2 3 4 5 6 7 8
│ │ │ │ │ │ │ │ │ └──────────┘
│ │ │ │ │ │ │ │ │     第 10-17 位：生产序列号
│ │ │ │ │ │ │ │ └─ 第 9 位：校验位
│ │ │ │ │ │ │ └─── 第 8 位：发动机类型
│ │ │ │ │ │ └───── 第 4-7 位：车型特征
│ │ │ │ └─────── 第 3 位：车辆类型
│ │ └───────── 第 2 位：制造商
└─────────── 第 1 位：生产国（L=中国）
```

### VIN 解读要点

- **第 1 位**：地理区域（L=中国，J=日本，W=德国，1/4/5=美国）
- **第 2-3 位**：制造商代码（FV=一汽大众，SV=上汽大众，HG=广汽本田）
- **第 10 位**：车型年份（M=2021，N=2022，P=2023，R=2024）

## 整车结构组成

汽车由四大系统构成：

```
┌──────────────────────────────────────┐
│              汽  车                   │
├──────────┬──────────┬───────┬────────┤
│  车身     │  底盘     │ 动力  │  电气   │
│  Body    │ Chassis  │ 总成  │ Electric│
├──────────┼──────────┼───────┼────────┤
│· 车身壳体 │· 传动系统 │· 发动机│· 电源   │
│· 车门    │· 行驶系统 │· 变速箱│· 起动机 │
│· 车窗    │· 转向系统 │       │· 照明   │
│· 内外饰  │· 制动系统 │       │· 仪表   │
└──────────┴──────────┴───────┴────────┘
```
