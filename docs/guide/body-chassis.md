# 车身与底盘

## 车身制造工艺

现代汽车车身制造遵循「四大工艺」流程：

```
冲压 → 焊接 → 涂装 → 总装
```

### 1. 冲压（Stamping）

将钢板通过冲压模具加工成车身覆盖件（车门、引擎盖、翼子板等）。

- **材料**：多为低碳钢板，厚度 0.6-0.8mm
- **趋势**：铝合金板材的使用比例持续上升（减重）

### 2. 焊接（Welding）

将冲压件焊接成白车身（Body in White, BIW）。

- **点焊**：最常用，一辆车约有 3000-5000 个焊点
- **激光焊**：用于车顶与侧围连接，焊缝美观
- **涂胶**：结构胶用于增强刚度和 NVH 性能

### 3. 涂装（Painting）

对白车身进行防腐处理和上色。

```
前处理（脱脂/磷化）→ 电泳（底漆）→ 中涂 → 面漆 → 清漆
```

- 电泳：整个车身浸入电泳池，形成均匀防锈层
- 机器人喷涂：面漆和清漆基本实现全自动化

### 4. 总装（Final Assembly）

将动力总成、底盘、内饰、电子设备等安装到车身上。

- 流水线作业，每站节拍约 60-90 秒
- 关键工序：挡风玻璃涂胶、底盘合装、车轮拧紧（扭矩控制）

## 车身材料

| 材料 | 特点 | 典型应用 |
|------|------|----------|
| **低碳钢** | 成本低、成型好 | 车身主体覆盖件 |
| **高强度钢（HSS）** | 强度高、安全 | A/B 柱、门槛梁、防撞梁 |
| **热成型钢** | 强度极高（1500MPa+） | B 柱加强板、车门防撞杆 |
| **铝合金** | 轻量化（密度约为钢 1/3） | 引擎盖、车门、副车架 |
| **碳纤维** | 超轻、超强、超贵 | 超跑车身、高端车饰件 |

## 底盘系统

底盘是汽车的骨架，包含四大子系统：

### 传动系统

将发动机动力传递到驱动轮。

发动机 → 离合器 → 变速箱 → 传动轴 → 差速器 → 半轴 → 车轮

### 行驶系统

支撑车身，缓冲路面冲击。

- 车架/承载式车身
- 悬架（弹簧 + 减震器）
- 车轮与轮胎

### 转向系统

控制车辆行驶方向。

- 方向盘 → 转向柱 → 转向机 → 转向拉杆 → 车轮
- 助力方式：机械 → 液压助力（HPS）→ 电动助力（EPS）

### 制动系统

使车辆减速或停止。

- 行车制动（脚刹）：液压制动
- 驻车制动（手刹/电子手刹）
- 辅助制动：发动机制动、缓速器

## 关键尺寸参数

| 参数 | 定义 | 影响 |
|------|------|------|
| **轴距** | 前后轮轴中心距 | 越大越稳，影响转弯半径 |
| **轮距** | 左右车轮中心距 | 越宽越稳，影响通过性 |
| **车长** | 前后最外端点距离 | 影响停车和城市通行 |
| **离地间隙** | 底盘最低点到地面距离 | 影响通过性 |
