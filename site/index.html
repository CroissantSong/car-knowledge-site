1 整车整体认知

目标：说清一辆车由哪些大系统组成，以及能量如何从能源传到车轮。

  • 组成：车身 / 底盘 / 动力总成 / 电气电子 / 内外饰
  • 能量链：能源 → 转换 → 传递 → 车轮
  • 四种架构一眼区分：燃油 / 混动 / 增程 / 纯电
  • 术语：平台、白车身(BIW)、三电、域控制器

前置：无 · 掌握标准：能画出整车系统框图并解释能量流

2 机械基础补课

目标：补齐理解后续系统所需的物理与机械概念。

  • 力、扭矩、功率、转速的关系
  • 齿轮与传动比：为什么要变速 / 减速
  • 摩擦、热量、效率：能量在哪损耗
  • 峰值功率 vs 持续功率（看参数的关键）

前置：第 1 层 · 掌握标准：看到参数表能解释每项含义

3 传统汽车系统（燃油车）

目标：理解从「烧油」到「驱动」的完整链路与底盘系统。

  • 发动机：四冲程、进排气、燃烧
  • 变速箱：MT / AT / CVT / DCT
  • 传动：离合、传动轴、差速器
  • 底盘：悬架、转向、制动；车身与轮胎

前置：第 2 层 · 掌握标准：能讲清「踩油门到车轮转动」

4 新能源汽车

目标：理解三电与新能源架构，并能与燃油车对比。

  • 电池 / 电机 / 电控（三电）；BMS、逆变器
  • 充电与补能；热管理
  • 架构：纯电(BEV) / 混动(HEV·PHEV) / 增程(EREV)

前置：第 2 层 · 掌握标准：能解释「纯电车为何不需要传统变速箱」

5 智能汽车基础

目标：理解让车「聪明」的电子电气与软件骨架。

  • 传感器：摄像头 / 毫米波 / 激光雷达 / 超声波
  • ECU 与域控制器；CAN 总线与车载网络
  • 线控（转向/制动）；ADAS 与分级；OTA

前置：第 1 层 · 掌握标准：能解释会议里的「域控」「CAN」

动力链对比 · 能量 → 车轮

本站的核心可视化：用同一条链路对比四种架构。

汽油→发动机→变速箱→车轮燃油 ICE
油+电→发动机+电机→耦合→车轮混动 HEV/PHEV
汽油→发动机(只发电)→电机→车轮增程 EREV
电池→电机(单速)→车轮纯电 BEV

看点：只有燃油/部分混动需要传统多挡变速箱；增程的发动机不直接驱动车轮。
